# Screenshots

This folder contains screenshots of the PlayTunes Music Player app.

## Recommended Screenshots

Please add the following screenshots to showcase the app:

### Main Screen
- `main_screen.png` - Song list view with album artwork
- `main_screen_with_controls.png` - Bottom control bar visible

### Now Playing Screen
- `now_playing.png` - Full-screen now playing view
- `now_playing_controls.png` - Playback controls highlighted

### UI States
- `play_state.png` - Play button active state
- `pause_state.png` - Pause button active state

## Image Guidelines

- **Format**: PNG preferred for UI screenshots
- **Resolution**: Use device screenshots (1080x1920 or similar)
- **Quality**: High quality, clear and readable
- **Naming**: Use descriptive, lowercase names with underscores
- **Size**: Keep individual files under 2MB for faster loading

## How to Add Screenshots

1. Take screenshots on your Android device/emulator
2. Save them in this folder with descriptive names
3. Update the main README.md to reference them
4. Commit and push to the repository
